# **App Name**: PaulMaker Stream

## Core Features:

- Intelligent Schedule Matrix: A dynamic 7-day visual calendar that tracks booking density and prevents overages beyond the 12-hour daily limit.
- Unified Admin Command Center: A Pterodactyl-inspired dashboard for total management of bookings, statistics, and live statuses.
- AI Promotion Suite: A generative tool used to brainstorm high-converting livestream titles and server descriptions based on GTPS characteristics.
- Cloud Record Keeper: Native Firestore implementation for real-time storage of schedules, statistics, and booking metadata.
- External Asset Bridge: Integration with Google Sheets and Google Drive for automated archival of business logs and testimonial imagery.
- Verified Client Gallery: Automated testimonial display that pulls assets directly from Drive for instant, public social proof.
- Lead Streamliner: Validation-heavy booking modal that exports clean data and redirects directly to WhatsApp for payment finalization.

## Style Guidelines:

- Primary: Gaming Red (#D90429), a vibrant and aggressive red inspired by esports energy. Background: Pitch Black (#050505) with 20% saturation to maintain deep inkiness while nodding to the primary hue. Accent: Sunfire Orange (#E85D04), providing high-contrast for CTA elements and highlights.
- Headline: 'Space Grotesk', a tech-heavy sans-serif. Body: 'Inter', an objective sans-serif for high readability in dense data panels.
- Outline-style gaming icons with subtle #D90429 glow filters for an interface that feels illuminated from within.
- Modular bento-box grid inspired by high-end server hosting panels, utilizing cards with sharp corners and subtle 1px #2A2A2A borders.
- High-speed card transitions and floating particle backgrounds to maintain a futuristic, fantasy atmosphere without impacting performance.
